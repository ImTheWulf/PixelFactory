# PF-0009 — Universal Image Viewer

Status: Implemented for testing

## Goal
Create one shared image viewing experience across Pixel Factory so generated images, assets, Palette Lab previews, and future tools all open the same large viewer.

## Included
- Full-screen modal viewer.
- Fit-to-window mode.
- Actual pixel / 1:1 mode.
- Zoom in / zoom out controls.
- Mouse wheel zoom.
- Drag/pan while zoomed.
- ESC closes the viewer.
- `0` switches to fit mode.
- `1` switches to actual-pixel mode.
- Download link inside viewer.
- Palette Lab preview sizing fixes to avoid image squashing.

## Notes
This is a framework commit. The viewer should be reused by future Tile Studio, Repair Bench, Exporter, and Aseprite/Godot preview tools.
