# PixelFactory App

Desktop application shell for deterministic pixel-art processing.

Planned stack:

- Python 3.12+
- PySide6 for UI
- Pillow for image operations
- NumPy/OpenCV later for analysis and cleanup

## Local setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
