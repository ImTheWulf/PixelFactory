# PF-0013.3 Export Selection Foundation

This pass adds explicit selected-asset export on top of the PF-0013 PNG-first export foundation.

## Added

- Backend batch endpoint:
  - `POST /api/exports/{target}/assets`
  - Body: `{ "asset_ids": ["..."] }`
- Export service method:
  - `ExportService.export_assets(asset_ids, target)`
  - Deduplicates selected IDs before export.
  - Returns exported records plus per-asset failures.
- Asset Browser UI:
  - Per-card `Export` checkbox.
  - Inspector `Select Export` / `Unselect Export` button.
  - Selected cards receive a visible border.
- Exporter UI:
  - `Export Selected Assets` button.
  - `Clear Selection` button.
  - Selected-count status.
  - Export status panel now shows selected count.

## Scope

PF-0013.3 keeps the export layer PNG-first. It does not add fake Godot or Aseprite deep integration yet. The goal is to make the current foundation more usable while leaving room for Godot resource helpers, sprite sheets, and Aseprite frame/tag work later.
