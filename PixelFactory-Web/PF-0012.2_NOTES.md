# PF-0012.2 Quick Favorite + Seed UI Cleanup

## Changes

- Fixed quick favorite toggle on asset cards.
- Asset cards now use a clickable star button: `☆` / `★`.
- Magnifying glass remains dedicated to opening the universal image viewer.
- Renamed the seed display concept to **Last Generated Seed**.
- Last Generated Seed stays hidden until a generation actually completes.
- Use Random Seed now fills only the next seed input, not the last-generated display.

## Intended behavior

- Click card = inspect/select.
- Click magnifier = view large.
- Click star = favorite/unfavorite.
- Seed field = next seed to use.
- Last Generated Seed = seed from the last completed generation.
