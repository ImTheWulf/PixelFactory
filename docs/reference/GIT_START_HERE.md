# Git / GitHub Desktop setup

## Simple path with GitHub Desktop

1. Install GitHub Desktop.
2. Open GitHub Desktop.
3. File → Add local repository.
4. Select this `PixelFactoryByWulf` folder.
5. If it asks to create a repository, say yes.
6. Commit message: `Initial Pixel Factory project scaffold`.
7. Click Commit.
8. Optional: Publish repository to GitHub.

## Important

Do not commit model files. The `.gitignore` blocks common model formats like `.safetensors`, `.gguf`, `.ckpt`, `.pt`.
