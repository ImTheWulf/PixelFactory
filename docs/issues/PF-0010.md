# PF-0010 — Asset Metadata + Inspector Polish

## Status
Implemented for testing.

## Goal
Make generated assets easier to organize, inspect, and reuse.

## Changes
- Editable asset name.
- Editable tags.
- Editable notes.
- Favorite / unfavorite toggle.
- Improved inspector sections.
- Generation metadata grid.
- File path display.
- Copy buttons for prompt, negative prompt, and resolved prompt.
- Backend metadata update endpoint.

## Test checklist
- Select an asset.
- Rename it and save metadata.
- Add tags and notes.
- Favorite / unfavorite it.
- Copy prompt and negative prompt.
- Accept an incoming asset and confirm Accept disappears.
- Refresh Asset Browser and confirm metadata persists.
