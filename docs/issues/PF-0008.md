# PF-0008 — Workspace Pipeline

## Status
Implemented for test build.

## Goal
Make Pixel Factory behave like an editor with a current working image instead of isolated pages that each manage their own file.

## Changes
- Added `WorkspaceService`.
- Added `/api/workspace` endpoints.
- Latest Character Studio generation becomes the current workspace image.
- Palette Lab can load the current workspace image without manual browsing.
- Asset Browser can set any asset as the current workspace.
- Page load now checks ComfyUI status automatically.
- Palette Lab added Fit / Actual Pixels preview mode.

## Test Plan
1. Start ComfyUI.
2. Start Pixel Factory web app.
3. Open the page and confirm Comfy status updates automatically.
4. Generate a character.
5. Switch to Palette Lab and confirm the generated image loads from workspace.
6. Open Asset Browser, select an asset, use Set Workspace and Palette Lab.
